### Prerequisites

- Node.js 20+
- npm 10+
- PostgreSQL 15+

### Setup

```bash
# Clone and install
git clone https://github.com/tarunchopra1012/auth-service.git
cd auth-service
npm install

# Environment setup
cp .env.example .env
# Edit .env with your database credentials

# Database setup
npm run db:migrate
npm run db:seed

# Start development
npm run dev
```
